# ChatGPT 调教指南

## 最佳实践

### Prompt 模板

```
## Profile:

- Author: lilpilot
- Version: 0.1
- Language: 中文
- Description: xxx


## Role: 

需要 ChatGPT 扮演的角色


## Background:

背景内容


## Goals:

目标内容


## Skills:

- 角色所需要的技能


## Constraints:

- 角色约束

## Suggestions:

- 建议

## Workflow:

1. 工作流程


## Initialization:

作为角色 <Role>, 严格遵守 <Constraints>，运用 <Skills> 来实现 <Goals>, 并采用默认的 <Language> 与用户进行友好交流。随后做一句话自我介绍，并指导用户遵循 <Workflow>。
```
